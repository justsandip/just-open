# JustOpen

**Stop digging through tabs, Start coding.**

JustOpen is a minimalist, local-first dashboard for open-source contributors. Stop jumping between 20 tabs to find your next contribution. JustOpen monitors the repositories you care about and filters them for the issues you actually want to solve.

![JustOpen Hero Shot](/assets/justopen-3.png)

## ✨ Key Features

* **Zero Infrastructure**: No accounts, no databases. Everything lives in your local storage.
* **Unified Feed**: Chronological aggregation of issues across multiple organizations.
* **One-Click Intent**: Pre-set filters for "Good first issues," "Unassigned bugs," and "Docs."
* **Privacy First**: Your data never leaves your browser.

## 🛠 How it Works

1. **Define Your Watchlist:** Input the `owner/repo` (e.g., `vercel/next.js`).
2. **Set Your Filter:** Use standard GitHub search syntax to target specific labels or states.
3. **Monitor & Contribute:** The feed updates instantly, giving you a direct bridge to the issue page.
